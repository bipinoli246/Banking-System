#  Banking System

A simple Python project that lets users create accounts, deposit and withdraw money, and check their balance.

##  Features
- Create new account  
- Deposit money  
- Withdraw money  
- Check account balance  

##  Tech Used
- Python  
- File Handling / SQL  

##  How to Run
1. Clone the repository  
2. Open the folder in your code editor  
3. Run the file
